Professional Git Flow CLI

Streamline your Git workflow with an elegant and powerful command-line interface. Feature branches, releases, and synchronization made simple.

npm install -g gitnull

Feature Management

Start and finish feature branches with ease

gitnull start-feature <name>
gitnull finish-feature <name>

Release Control

Manage releases with version control

gitnull start-release <version>
gitnull finish-release <version>

Sync Operations

Keep your repository synchronized

gitnull push
gitnull sync
